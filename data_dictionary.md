# Data Dictionary – Data Professional Survey

This document explains each column used in the cleaned dataset.

| Column Name | Description |
|------------|-------------|
| Unique ID | Unique identifier for each survey response |
| Email | Respondent email (anonymized / not used in analysis) |
| Date Taken | Date when the survey was submitted |
| Time Taken | Time when the survey was submitted |
| Time Spent | Duration spent completing the survey |
| Q1 - Current Role | Current job title of the respondent |
| Q2 - Career Switch | Whether the respondent switched careers into data |
| Average Salary | Average of lower and upper salary range (USD) |
| Q4 - Industry | Industry in which the respondent works |
| Q5 - Programming Language | Preferred programming language |
| Q6 - Salary Satisfaction | Satisfaction with salary (1–5) |
| Q6 - Work/Life Balance | Satisfaction with work-life balance (1–5) |
| Q6 - Coworkers | Satisfaction with coworkers (1–5) |
| Q6 - Management | Satisfaction with management (1–5) |
| Q6 - Upward Mobility | Satisfaction with growth opportunities (1–5) |
| Q6 - Learning | Satisfaction with learning opportunities (1–5) |
| Q7 - Difficulty | Difficulty breaking into data roles |
| Q8 - Job Priority | Most important factor when looking for a new job |
| Q9 - Gender | Gender of respondent |
| Q10 - Age | Age of respondent |
| Q11 - Country | Country of residence |
| Q12 - Education | Highest education level |
| Q13 - Ethnicity | Ethnicity of respondent |
